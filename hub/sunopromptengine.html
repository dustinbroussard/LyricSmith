      About
      
    

    

      
Suno Prompt Engine is a mobile‑first, light/dark, production‑ready web app that crafts Suno v4.5 prompts using OpenRouter models. Nothing leaves your browser except direct API calls you make with your key (stored locally). You can edit the system prompt and export the outputs.

      
    
        
  • Enter your OpenRouter API key in Settings.
    • 
        
  • Fetch and pick a model (e.g., gpt‑4.1‑mini, llama‑3.1‑70b, etc.).
    • 
        
  • Describe your track, hit Generate.
    • 
        
  • Copy each output or export a combined .txt.
    • 
      

      
Security: API key is kept in localStorage. Clear it any time in Settings. OpenRouter requests are sent to https://openrouter.ai/api/v1.
